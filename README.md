# VRCMoreRefresh
Adds more refresh buttons to VRChat.

## Current refresh button locations
> - Social Screen
> - User Info Screen

## Requirements
- [VRCModLoader by Slaynash](https://github.com/Slaynash/VRCModLoader)
- [VRCMenuUtils](https://github.com/AtiLion/VRCMenuUtils)
